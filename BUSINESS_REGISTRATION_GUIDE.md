# 사업자 등록 필요성 가이드

## 결론: 무조건 필요하지 않음

개인 개발자도 수익을 낼 수 있습니다. 다만 상황에 따라 고려해야 할 사항이 있습니다.

---

## 1. Paddle 요구사항

### 개인 개발자도 사용 가능 ✅
- Paddle은 개인 개발자(Sole Proprietorship)도 지원합니다
- 사업자 등록 없이도 결제 처리 가능
- 다만, 일정 수익 이상이면 추가 검증을 요구할 수 있습니다

### 언제 사업자 등록이 필요할 수 있나?
- **월 수익 $10,000 이상**: 일부 결제 처리업체가 요구할 수 있음
- **연 수익 $100,000 이상**: 세금 신고 시 사업자 등록이 유리할 수 있음
- **법적 책임 분리 필요**: 개인 자산 보호가 필요한 경우

---

## 2. 세금 문제 (한국 기준)

### 개인 소득으로 처리 (사업자 등록 없음)
- **장점**: 간단함, 추가 비용 없음
- **단점**: 
  - 종합소득세로 과세 (최대 45%)
  - 비용 공제 제한적
  - 연간 120만원 이상이면 부가세 신고 필요

### 사업자 등록 (개인사업자)
- **장점**:
  - 사업 소득으로 분리 과세
  - 비용 공제 가능 (서버비, 도메인, API 비용 등)
  - 부가세 면세 사업자 가능 (연 매출 4,800만원 미만)
- **단점**:
  - 사업자 등록 비용 (무료)
  - 분기별 부가세 신고 필요 (면세 사업자는 간이 신고)

### 추천 기준
- **월 수익 $500 미만**: 개인 소득으로 처리 (사업자 등록 불필요)
- **월 수익 $500-2,000**: 선택적 (세금 절감 효과 고려)
- **월 수익 $2,000 이상**: 사업자 등록 권장 (비용 공제 효과 큼)

---

## 3. 법적 책임

### 개인 책임 (사업자 등록 없음)
- 개인 자산으로 모든 책임을 집니다
- 소송 시 개인 재산까지 위험

### 법인 책임 (사업자 등록)
- 사업 재산으로만 책임을 집니다
- 개인 자산 보호 가능

### Cost Signal의 경우
- 정보 제공 서비스 (금융 조언 아님)
- 법적 리스크가 낮은 편
- 초기에는 개인 책임으로도 충분할 수 있음

---

## 4. 실전 추천 전략

### Phase 1: 초기 (월 수익 $500 미만)
```
✅ 사업자 등록 불필요
✅ 개인 소득으로 처리
✅ Paddle 개인 개발자 계정 사용
✅ 간단하게 시작
```

### Phase 2: 성장기 (월 수익 $500-2,000)
```
⚠️ 사업자 등록 고려
- 비용 공제 효과 계산
- 세금 절감 vs 관리 비용 비교
- 필요시 개인사업자 등록
```

### Phase 3: 안정화 (월 수익 $2,000 이상)
```
✅ 사업자 등록 권장
- 비용 공제로 세금 절감
- 법적 책임 분리
- 더 전문적인 이미지
```

---

## 5. 한국 개인사업자 등록 방법 (필요시)

### 등록 조건
- 만 18세 이상
- 사업자 등록 신청서 제출
- 등록 비용: **무료**

### 등록 절차
1. **사업자등록 신청**: 국세청 홈택스 (https://www.hometax.go.kr)
2. **신청서 작성**: 사업자 정보, 업종 코드 입력
3. **제출 및 승인**: 보통 1-2일 내 승인

### 업종 코드
- **소프트웨어 개발**: 62010 (컴퓨터 프로그래밍업)
- **정보 제공 서비스**: 63910 (기타 정보 서비스업)

### 비용
- 등록 비용: **무료**
- 유지 비용: 없음 (부가세 면세 사업자는 간이 신고만)

---

## 6. Paddle 답변 전략

### 현재 상황 (사업자 등록 없음)
```
✅ 개인 개발자로 답변
✅ 필요시 향후 등록 가능하다고 명시
✅ Paddle은 개인 개발자도 지원하므로 문제없음
```

### 향후 계획
- 수익이 늘어나면 사업자 등록 고려
- Paddle이 요구하면 등록 진행
- 세금 절감 효과가 크면 등록

## 7. 결론 및 추천

### 지금 당장은?
✅ **사업자 등록 불필요**
- 개인 개발자로 Paddle 사용 가능
- 초기에는 간단하게 시작
- 수익이 늘어나면 나중에 등록 가능

### 언제 등록해야 하나?
- **월 수익 $2,000 이상**이 되면 고려
- **비용 공제 효과**가 관리 비용보다 클 때
- **법적 책임 분리**가 필요할 때

### Cost Signal의 경우
- 초기 목표: 1000명 구독자 = 월 $4,990
- 이 수준이면 **사업자 등록 권장**
- 하지만 지금 당장은 불필요

## 참고사항

1. **Paddle은 개인 개발자도 지원**: 지금 당장 사업자 등록 없이도 시작 가능
2. **나중에 등록 가능**: 수익이 늘어나면 언제든지 등록 가능
3. **세금 전문가 상담**: 수익이 늘어나면 세무사와 상담 권장
4. **국가별 차이**: 한국과 미국의 세금 체계가 다르므로 주의
