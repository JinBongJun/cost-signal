  - 제3자 서비스 (Paddle, Sentry, Google OAuth) 설명
  - 사용자 권리 명시
  - 쿠키 및 추적 정책
- ✅ **Footer 링크** 추가
  - 이용약관 (`/terms`) 링크
  - 개인정보처리방침 (`/privacy`) 링크

### 2. FAQ 페이지
- ✅ **FAQ 페이지** (`/faq`) 추가
  - 10개의 자주 묻는 질문
  - 접기/펼치기 인터랙션
  - 홈페이지에서 FAQ 링크 추가
  - 질문 카테고리:
    - 서비스 소개
    - 신호 계산 방법
    - 업데이트 주기
    - 금융 조언 여부
    - 데이터 소스
    - 무료/유료 차이
    - 구독 취소
    - 푸시 알림
    - 데이터 보안
    - 비밀번호 찾기

### 3. 에러 처리 개선
- ✅ **오프라인 상태 감지**
  - `navigator.onLine` 체크
  - 오프라인 시 명확한 메시지 표시
  - 오프라인 상태 알림 배너
- ✅ **에러 메시지 개선**
  - 더 구체적인 에러 분류
  - 구독 필요 에러 분리
  - 서버 에러와 네트워크 에러 구분
  - 각 에러에 대한 해결 방법 제시
- ✅ **Toast 알림 통합**
  - 네트워크 에러 시 Toast 표시
  - 사용자 친화적 피드백

### 4. 접근성 개선
- ✅ **키보드 네비게이션**
  - 모든 버튼에 `focus:ring` 스타일 추가
  - 포커스 표시 명확화
- ✅ **ARIA 속성**
  - 버튼에 `aria-label` 추가
  - 오프라인 알림에 `role="alert"` 추가
  - FAQ 토글 버튼에 `aria-label` 추가
- ✅ **시각적 피드백**
  - 포커스 링 스타일 통일
  - 버튼 호버 효과 개선

### 5. 모바일 최적화
- ✅ **터치 타겟 크기**
  - 최소 44x44px 터치 타겟 보장
  - 모바일에서 버튼 크기 최적화
- ✅ **iOS 최적화**
  - 입력 필드 자동 확대 방지 (`font-size: 16px`)
  - 뷰포트 메타 태그 추가
  - 사용자 확대 허용 (`user-scalable=yes`)
- ✅ **반응형 개선**
  - 모바일에서 더 나은 레이아웃
  - 터치 제스처 지원

### 6. 성능 및 UX
- ✅ **애니메이션 최적화**
  - `prefers-reduced-motion` 지원
  - 모션 감소 설정 시 애니메이션 비활성화
- ✅ **로딩 상태**
  - 오프라인 상태 명확히 표시
  - 네트워크 에러 시 재시도 옵션 제공
