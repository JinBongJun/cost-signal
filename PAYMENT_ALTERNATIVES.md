# 결제 시스템 대안

## 현재 상황
- 목표: 유료 사용자 1000명
- 문제: Stripe가 복잡함 (법인 등록 필요)

## 대안 옵션

### 옵션 1: Paddle (추천) ⭐
**장점**:
- 한국에서 사용 가능
- 법인 등록 불필요 (개인도 가능)
- 세금 자동 처리
- USD 결제 지원
- 설정 간단

**단점**:
- 수수료 약간 높음 (5% + $0.50)
- Stripe보다 덜 알려짐

**비용**: $4.99 결제 시 약 $0.75 수수료

### 옵션 2: LemonSqueezy
**장점**:
- 한국 지원
- 설정 간단
- 세금 자동 처리

**단점**:
- 수수료: 3.5% + $0.30

### 옵션 3: Gumroad
**장점**:
- 매우 간단
- 개인도 사용 가능

**단점**:
- SaaS 구독보다 제품 판매에 적합
- 커스터마이징 제한

### 옵션 4: 무료로 시작 → 나중에 결제 추가
**전략**:
1. 무료 서비스로 시작
2. 사용자 확보 및 검증
3. 충분한 사용자 확보 후 결제 추가

**장점**:
- 즉시 시작 가능
- 사용자 피드백 수집
- 검증 후 결제 추가

**단점**:
- 초기 수익 없음
- 나중에 결제 전환 필요

## 추천: Paddle 사용

### 이유
1. 한국에서 사용 가능
2. 법인 등록 불필요
3. 설정 간단
4. USD 결제 지원
5. 세금 자동 처리

### Paddle vs Stripe
| 항목 | Paddle | Stripe |
|------|--------|--------|
| 한국 지원 | ✅ | ⚠️ (복잡) |
| 법인 필요 | ❌ | ✅ (미국) |
| 설정 난이도 | 쉬움 | 복잡 |
| 수수료 | 5% + $0.50 | 2.9% + $0.30 |
| 세금 처리 | 자동 | 수동 |

**Paddle 수수료가 높지만**, 법인 등록 비용과 시간을 고려하면 더 나을 수 있습니다.
