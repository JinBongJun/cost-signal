## ❌ 부족한 기능 (실제 서비스 비교)

### 🔴 높은 우선순위 (사용자 경험 필수)

#### 1. **다크모드 토글 버튼**
- **현재**: 시스템 설정만 따름 (사용자가 직접 변경 불가)
- **필요**: 헤더에 다크모드 토글 버튼 추가
- **예시**: Notion, GitHub, Linear 등 모든 주요 서비스
- **구현 시간**: 30분

#### 2. **이메일 알림 설정**
- **현재**: 푸시 알림만 있음
- **필요**: 이메일 알림 설정 페이지 (주간 요약, 중요 알림 등)
- **예시**: 모든 SaaS 서비스
- **구현 시간**: 1-2시간

#### 3. **구독 업그레이드/다운그레이드**
- **현재**: 구독 취소만 가능
- **필요**: 플랜 변경 기능 (Monthly ↔ Yearly ↔ Early Bird)
- **예시**: 모든 구독 서비스
- **구현 시간**: 2-3시간

#### 4. **사용자 피드백 시스템**
- **현재**: 없음
- **필요**: 피드백 제출 폼, 버그 리포트, 기능 요청
- **예시**: 모든 서비스
- **구현 시간**: 1-2시간
