### 🟡 중간 우선순위 (사용자 경험)

#### 2. Free/Paid 티어 재조정
**상태**: 분석 완료, 구현 필요  
**우선순위**: 중간  
**예상 시간**: 2-3시간  
**설명**: 현재 Free 티어가 너무 좋아서 유료 전환 동기가 약함

**현재 상태**:
- Free: 신호 + 설명 (핵심 가치 충족)
- Paid: 신호 + 설명 + 지표 상세 + 히스토리

**제안된 변경**:
- Free: 신호만 (설명 없음)
- Paid: 신호 + 설명 + 지표 상세 + 히스토리

**효과**: 전환율 1-2% → 5-8% 예상

**필요한 작업**:
- `/api/signal` 엔드포인트 수정 (Free 티어는 설명 제외)
- UI에서 Free 사용자에게 설명 숨기기
- "Upgrade to see explanation" CTA 추가
