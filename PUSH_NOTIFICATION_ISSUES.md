# 푸시 알림 및 PWA 문제 해결

## 현재 문제

1. **알림 권한 팝업이 나타나지 않음**
   - "Requesting notification permission..." 로그는 나오지만
   - 브라우저 알림 권한 팝업이 나타나지 않음

2. **PWA 설치 필요**
   - 앱으로 설치해야 함
   - 모바일에서도 작동해야 함

---

## 문제 원인 분석

### 알림 권한 팝업이 안 뜨는 이유

1. **브라우저가 팝업을 차단**
   - 일부 브라우저는 자동 팝업을 차단
   - 사용자 상호작용(버튼 클릭)이 필요

2. **이미 권한이 설정됨**
   - 브라우저 설정에서 이미 "차단" 또는 "허용"으로 설정됨
   - "default" 상태여도 브라우저가 팝업을 표시하지 않을 수 있음

3. **HTTPS 필요**
   - Production에서는 HTTPS 필수
   - Vercel은 자동으로 HTTPS 제공

---

## 해결 방법

### 1. 브라우저 설정 확인

**Chrome/Edge:**
1. 주소창 왼쪽의 자물쇠 아이콘 클릭
2. "사이트 설정" 클릭
3. "알림" 섹션 확인
4. "차단"이면 "허용"으로 변경

**또는:**
1. 브라우저 설정 → 개인정보 및 보안 → 사이트 설정 → 알림
2. 해당 사이트 찾기
3. 권한 변경

---

### 2. PWA 설치 방법

**데스크톱 (Chrome/Edge):**
1. 주소창 오른쪽의 설치 아이콘 클릭
2. 또는 "📱 Install App" 버튼 클릭

**모바일 (Android):**
1. Chrome에서 사이트 접속
2. 메뉴 → "홈 화면에 추가" 또는 "앱 설치"
3. 또는 자동으로 설치 팝업 표시

**모바일 (iOS):**
1. Safari에서 사이트 접속
2. 공유 버튼 → "홈 화면에 추가"

---

## 테스트 방법

### 알림 권한 수동 확인

브라우저 콘솔에서 실행:
```javascript
// 현재 권한 상태 확인
console.log('Permission:', Notification.permission);

// 권한 요청 (팝업이 나타나야 함)
Notification.requestPermission().then(permission => {
  console.log('Result:', permission);
});
```
