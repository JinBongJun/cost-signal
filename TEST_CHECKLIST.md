# 기능 테스트 체크리스트

## 📋 테스트 환경

**테스트 URL**: `https://cost-signal.com`

**브라우저**: Chrome, Edge (각각 테스트)

---

## 1. 인증 (Authentication) 🔐

### 1.1 회원가입
- [ ] 이메일 회원가입
  - [ ] 유효한 이메일 형식 입력
  - [ ] 비밀번호 입력 (8자 이상)
  - [ ] 이름 입력 (선택사항)
  - [ ] 회원가입 성공 확인
  - [ ] 자동 로그인 확인
- [ ] 에러 케이스
  - [ ] 중복 이메일로 회원가입 시도 → 에러 메시지 표시
  - [ ] 잘못된 이메일 형식 → 에러 메시지 표시
  - [ ] 짧은 비밀번호 (8자 미만) → 에러 메시지 표시
  - [ ] 빈 필드 제출 → 에러 메시지 표시

### 1.2 로그인
- [ ] 이메일 로그인
  - [ ] 등록된 이메일로 로그인
  - [ ] 올바른 비밀번호 입력
  - [ ] 로그인 성공 확인
  - [ ] 메인 페이지로 리디렉션 확인
- [ ] Google 로그인
  - [ ] "Sign in with Google" 버튼 클릭
  - [ ] Google 계정 선택
  - [ ] 로그인 성공 확인
  - [ ] 메인 페이지로 리디렉션 확인
- [ ] 에러 케이스
  - [ ] 잘못된 이메일 → 에러 메시지 표시
  - [ ] 잘못된 비밀번호 → 에러 메시지 표시
  - [ ] 빈 필드 제출 → 에러 메시지 표시

### 1.3 로그아웃
- [ ] 로그아웃 버튼 클릭
- [ ] 세션 종료 확인
- [ ] 로그인 페이지로 리디렉션 확인

### 1.4 비밀번호 재설정
- [ ] 비밀번호 찾기
  - [ ] "Forgot Password" 링크 클릭
  - [ ] 등록된 이메일 입력
  - [ ] 재설정 이메일 전송 확인
  - [ ] 이메일에서 링크 클릭
  - [ ] 새 비밀번호 입력
  - [ ] 비밀번호 재설정 성공 확인
  - [ ] 새 비밀번호로 로그인 확인
- [ ] 에러 케이스
  - [ ] 등록되지 않은 이메일 → 에러 메시지 표시
  - [ ] 만료된 토큰 → 에러 메시지 표시

## 3. 신호 데이터 (Signal Data) 📊

### 3.1 메인 페이지 (홈)
- [ ] Free 티어 (비로그인)
  - [ ] 신호 상태 표시 확인 (OK/CAUTION/RISK)
  - [ ] 날짜 표시 확인
  - [ ] 설명 없음 확인 (Free 티어)
  - [ ] 지표 상세 없음 확인 (Free 티어)
  - [ ] "Subscribe Now" 버튼 표시 확인
- [ ] Paid 티어 (구독 중)
  - [ ] 신호 상태 표시 확인
  - [ ] 설명 표시 확인
  - [ ] 지표 상세 표시 확인 (Gas, CPI, Interest Rate, Unemployment)
  - [ ] 각 지표의 상태 표시 확인
- [ ] 로딩 상태
  - [ ] 초기 로딩 스켈레톤 표시 확인
  - [ ] 데이터 로드 후 표시 확인
- [ ] 에러 상태
  - [ ] 네트워크 에러 시 에러 메시지 표시 확인

### 3.2 히스토리 (History)
