# 브라우저 알림 설정 가이드

## 현재 상황

알림 권한 팝업이 나타나지 않습니다. 브라우저 설정에서 직접 알림을 허용해야 합니다.

---

## Chrome/Edge 브라우저 설정 방법

### 방법 1: 주소창에서 설정 (가장 빠름)

1. **주소창 왼쪽의 자물쇠 아이콘 클릭**
   - 또는 정보 아이콘 (i) 클릭

2. **"사이트 설정" 클릭**
   - 또는 "권한" → "알림" 클릭

3. **"알림" 섹션 찾기**
   - 현재 상태 확인 (차단/허용/묻기)

4. **"허용"으로 변경**
   - 드롭다운에서 "허용" 선택

5. **페이지 새로고침**
   - Ctrl+Shift+R (하드 리프레시)

---

### 방법 2: 브라우저 설정에서 변경

**Chrome:**
1. 브라우저 오른쪽 상단의 **3점 메뉴 (⋮)** 클릭
2. **"설정"** 클릭
3. 왼쪽 사이드바에서 **"개인정보 및 보안"** 클릭
4. **"사이트 설정"** 클릭
5. **"알림"** 클릭
6. **"허용" 섹션**에서 사이트 찾기
   - 또는 **"차단" 섹션**에서 사이트를 찾아 **"허용"으로 이동**

**Edge:**
1. 브라우저 오른쪽 상단의 **3점 메뉴 (⋯)** 클릭
2. **"설정"** 클릭
3. 왼쪽 사이드바에서 **"쿠키 및 사이트 권한"** 클릭
4. **"알림"** 클릭
5. **"허용" 섹션**에서 사이트 찾기
   - 또는 **"차단" 섹션**에서 사이트를 찾아 **"허용"으로 이동**

---

### 방법 3: 주소창에 직접 입력

1. 주소창에 다음 입력:
   ```
   chrome://settings/content/notifications
   ```
   (Edge는 `edge://settings/content/notifications`)

2. **"허용" 섹션**에서 사이트 찾기
   - 없으면 **"차단" 섹션** 확인
   - 사이트를 찾아 **"허용"으로 변경**

---

## Firefox 브라우저 설정

1. 주소창 왼쪽의 **자물쇠 아이콘** 클릭
2. **"권한"** → **"알림"** 클릭
3. **"허용"** 선택
4. 페이지 새로고침

---

## Safari 브라우저 설정 (Mac)

1. Safari → **"환경 설정"** (Preferences)
2. **"웹사이트"** 탭 클릭
3. 왼쪽에서 **"알림"** 선택
4. 사이트 찾아서 **"허용"** 선택

---

## 모바일 브라우저 설정

### Android (Chrome)

1. Chrome 앱 열기
2. 메뉴 (3점) → **"설정"**
3. **"사이트 설정"** → **"알림"**
4. 사이트 찾아서 **"허용"** 선택

### iOS (Safari)

1. 설정 앱 열기
2. **"Safari"** → **"웹사이트"** → **"알림"**
3. 사이트 찾아서 **"허용"** 선택

---

## 확인 방법

### 브라우저 콘솔에서 확인

브라우저 개발자 도구 (F12) → Console 탭에서 실행:

```javascript
console.log('Notification permission:', Notification.permission);
```

결과:
- `"granted"` → ✅ 허용됨
- `"denied"` → ❌ 거부됨 (설정에서 변경 필요)
- `"default"` → ⚠️ 아직 요청 안 함

---

## 설정 후 다시 시도

1. **브라우저 설정에서 알림 허용**
2. **페이지 새로고침** (Ctrl+Shift+R)
3. **"🔔 Enable Weekly Notifications" 버튼 클릭**
4. 이제 작동해야 합니다!

---

## 문제 해결

### 여전히 작동하지 않으면

1. **브라우저 재시작**
2. **캐시 삭제**
   - Ctrl+Shift+Delete → 캐시된 이미지 및 파일 삭제
3. **시크릿 모드에서 테스트**
   - 확장 프로그램이 방해할 수 있음

---

## 완료 확인

설정 후 콘솔에서 확인:
```javascript
Notification.permission
// "granted"가 나오면 성공!
```

이제 푸시 알림이 작동할 것입니다! 🎉
